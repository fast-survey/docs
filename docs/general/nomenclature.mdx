|Name|Description|
|-|-|
|Configuration|The JSON file that defines the schema and content of a survey|
|User|The person that fills out a survey|
|Admin|The person that creates a survey|
|Admin Name|The unique name of an admin e.g. `max-mustermann`|
|Survey Name|The for one admin unique name of a survey e.g. `how-are-you-today`|
|Survey ID|The unique identifier of a survey, obtained by combining user name and survey name e.g. `max-mustermann.how-are-you-today`|
|Field|A component of the survey/configuration e.g. an Option field|
