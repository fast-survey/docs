|Name|Description|
|-|-|
|Configuration|The JSON file that defines the schema and content of a survey|
|Respondent|The person that fills out a survey|
|User|The person that creates a survey|
|Username|The unique name of a user e.g. `max-mustermann`|
|Survey Name|The per user unique name of a survey e.g. `how-are-you-today`|
|Survey ID|The unique identifier of a survey, obtained by combining username and survey name e.g. `max-mustermann.how-are-you-today`|
|Field|A component of the survey/configuration e.g. an Option field|
