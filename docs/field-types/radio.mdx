The Radio field provides choice between multiple Options. For the input to be valid, exactly one Option needs to be selected.

## Properties

- `fields`: list of Option fields

## Example JSON Specification:

```json
{
    "type": "Radio",
    "title": "What is your favorite fruit?",
    "description": "",
    "properties": {
        "fields": [
            {
                "type": "Option",
                "title": "Asparagus",
                "description": ""
            },
            {
                "type": "Option",
                "title": "Spinach",
                "description": ""
            },
            {
                "type": "Option",
                "title": "Artichoke",
                "description": ""
            },
            {
                "type": "Option",
                "title": "None of these is a fruit",
                "description": ""
            }
        ]
    }
}
```
