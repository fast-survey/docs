The Radio field provides choice between multiple Options. For the input to be valid, exactly one Option needs to be selected.

## Properties

- `title`: the title of the question
- `description`: a short optional description containing more details
- `fields`: list of Option fields

## Example JSON Specification:

```json
{
    "admin_name": "fastsurvey",
    "survey_name": "radio",
    "title": "Radio Test",
    "description": "",
    "start": 1000000000,
    "end": 2000000000,
    "draft": false,
    "authentication": "open",
    "limit": 0,
    "fields": [
        {
            "type": "radio",
            "title": "What is your favorite fruit?",
            "description": "",
            "fields": [
                {
                    "type": "option",
                    "title": "Asparagus",
                    "description": "",
                    "required": false
                },
                {
                    "type": "option",
                    "title": "Spinach",
                    "description": "",
                    "required": false
                },
                {
                    "type": "option",
                    "title": "Artichoke",
                    "description": "",
                    "required": false
                },
                {
                    "type": "option",
                    "title": "None of these is a fruit",
                    "description": "",
                    "required": false
                }
            ]
        }
    ]
}
```

## Example of a Valid Submission
