The Radio field provides choice between multiple Options. For the input to be valid, exactly one Option needs to be selected.

## Properties

- `fields`: list of Option fields

## Example JSON Specification:

```json
{
    "admin_name": "fastsurvey",
    "survey_name": "radio",
    "title": "Radio Test",
    "description": "",
    "start": 1000000000,
    "end": 2000000000,
    "mode": 0,
    "fields": [
        {
            "type": "Radio",
            "title": "What is your favorite fruit?",
            "description": "",
            "fields": [
                {
                    "type": "Option",
                    "title": "Asparagus",
                    "description": "",
                    "mandatory": false
                },
                {
                    "type": "Option",
                    "title": "Spinach",
                    "description": "",
                    "mandatory": false
                },
                {
                    "type": "Option",
                    "title": "Artichoke",
                    "description": "",
                    "mandatory": false
                },
                {
                    "type": "Option",
                    "title": "None of these is a fruit",
                    "description": "",
                    "mandatory": false
                }
            ]
        }
    ]
}
```
