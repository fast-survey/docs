The Option field provides a simple yes/no choice. It can be used on its own or as subfield to Radio and Selection fields.

## Properties

- `title`: the title of the question
- `description`: a short optional description containing more details
- `mandatory`: true if the field is mandatory to be selected, false otherwise

## Example JSON Specification:

```json
{
    "admin_name": "fastsurvey",
    "survey_name": "option",
    "title": "Option Test",
    "description": "",
    "start": 1000000000,
    "end": 2000000000,
    "mode": 0,
    "fields": [
        {
            "type": "Option",
            "title": "I have read and agree to the terms and conditions",
            "description": "",
            "mandatory": true
        }
    ]
}
```

## Example of a Valid Submission

```json
{
    "1": true
}
```
