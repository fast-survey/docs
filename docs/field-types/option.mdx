The Option field provides a simple yes/no choice. It can be used on its own or as subfield to Radio and Selection fields.

## Properties

- `title`: the title of the question
- `description`: a short optional description containing more details
- `required`: true if the field has to be selected, false otherwise

## Rules

- `title [string]`: `1 <= title.length <= 128`
- `description [string]`: `0 <= description.length <= 4096`
- `required [boolean]`

## Example JSON Specification:

```json
{
    "survey_name": "option",
    "title": "Option Test",
    "description": "",
    "start": 0,
    "end": 4102444800,
    "draft": false,
    "authentication": "open",
    "limit": 0,
    "fields": [
        {
            "type": "option",
            "title": "I have read and agree to the terms and conditions",
            "description": "",
            "required": true
        }
    ]
}
```

## Example of a Valid Submission

```json
{
    "1": true
}
```
