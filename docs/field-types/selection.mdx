The Selection field provides choice between multiple Options. You can specify the required number of Options to be selected.

## Properties

- `min_select`: the minimum number of Options to be selected (inclusive)
- `max_select`: the maximum number of Options to be selected (inclusive)
- `fields`: list of Option fields

## Example JSON Specification:

```json
{
    "admin_name": "fastsurvey",
    "survey_name": "selection",
    "title": "Selection Test",
    "description": "",
    "start": 1000000000,
    "end": 2000000000,
    "mode": 0,
    "fields": [
        {
            "type": "Selection",
            "title": "What are your favorite ice cream flavors?",
            "description": "",
            "min_select": 0,
            "max_select": 2,
            "fields": [
                {
                    "type": "Option",
                    "title": "Strawberry",
                    "description": "",
                    "mandatory": false
                },
                {
                    "type": "Option",
                    "title": "Vanilla",
                    "description": "",
                    "mandatory": false
                },
                {
                    "type": "Option",
                    "title": "Chocolate",
                    "description": "",
                    "mandatory": false
                }
            ]
        }
    ]
}
```
