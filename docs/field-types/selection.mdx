The Selection field provides choice between multiple Options. You can specify the required number of Options to be selected.

## Properties

- `title`: the title of the question
- `description`: a short optional description containing more details
- `min_select`: the minimum number of Options to be selected (inclusive)
- `max_select`: the maximum number of Options to be selected (inclusive)
- `fields`: list of Option fields

## Rules

- `title [string]`: `1 <= title.length <= 128`
- `description [string]`: `0 <= description.length <= 4096`
- `min_select [integer]`: `0 <= min_select <= max_select`
- `max_select [integer]`: `min_select <= max_select <= fields.length`
- `fields [list]`: `2 <= fields.length <= 128`

## Example JSON Specification:

```json
{
    "survey_name": "selection",
    "title": "Selection Test",
    "description": "",
    "start": 0,
    "end": 4102444800,
    "draft": false,
    "authentication": "open",
    "limit": 0,
    "fields": [
        {
            "type": "selection",
            "title": "What are your favorite ice cream flavors?",
            "description": "",
            "min_select": 0,
            "max_select": 2,
            "fields": [
                {
                    "type": "option",
                    "title": "Strawberry",
                    "description": "",
                    "required": false
                },
                {
                    "type": "option",
                    "title": "Vanilla",
                    "description": "",
                    "required": false
                },
                {
                    "type": "option",
                    "title": "Chocolate",
                    "description": "",
                    "required": false
                }
            ]
        }
    ]
}
```
